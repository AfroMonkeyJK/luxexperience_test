## Dependencies

Install the necessary dependencies:

```bash
npm install
npm init playwright@latest
npm install --save-dev @eslint/js
npm install winston
npm install dotenv
npm install cross-env
npm install multiple-cucumber-html-reporter
npm install rimraf
```
